== Description ==
Hide text/link/image in specific country.
 

== Installation ==

1.  Upload country-ban-system folder to plugins folder.

2.  Go to Plugins menu and active.

3. Add shortcode: [cbs_block_country][/cbs_block_country] to post or page.

Shortcode example: [cbs_block_country ct="LT,UK,US" message="Post not found"]Secret text[/cbs_block_country]
